WordPress Themes 101

About Themes

Themes are the basic presentational packages for WordPress. They include:

  • Templates

  • Styles

  • Basic Javascript

  • Supporting PHP functions

Types of Themes

There are two types of WordPress themes:

  • Full themes

  • Child Themes

Full Themes

Required Files

  • index.php

  • style.css

All other theme files add additional functionality.

functions.php

Loaded on every request.

This is where you can add functionality to WordPress surrounding your theme, or load the filed that do that.

  • Image sizes

  • Load scripts & styles

  • Custom headers

Templates

Template Hierarchy

How WordPress loads templates based on what’s available:

  • Based on the type of request

  • First matching template wins

  • Specific to general

  • Final default is always index.php

Child Themes

What is a Child Theme?

Child themes override parts of a parent theme, to build on and customize it to their needs.

  • styles

  • templates (usually header, footer)

  • functions.php - loaded in addition to parent theme

The only required file for a child theme is style.css.

Starter Themes

What’s a starter theme?

Starter themes are prebuilt, bare themes that provide a starting ground for building a custom WordPress theme:

  • Very few styles

  • Basic templates

  • Helper functions

Examples of Starter Themes

  • _s

  • Roots

  • HTML5 Blank

  • Handcrafted

  • Naked
